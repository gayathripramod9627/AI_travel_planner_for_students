# AI Travel Planner for Students

This project is a simple AI-based travel planner designed for students with limited budgets. 
It generates personalized, budget-friendly travel itineraries based on destination, number of days, and total budget.

## Features
- Student budget-based travel planning
- Day-wise itinerary generation
- Affordable activities, food, and transport suggestions
- Simple Flask web application
- Single Python file implementation

## Technologies Used
- Python
- Flask
- HTML & CSS (embedded)
- Rule-based AI logic
